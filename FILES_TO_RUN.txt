Problem: "Cannot connect to backend"
Solution: Make sure backend is running: python app.py

Problem: "GROQ_API_KEY not configured"
Solution: Add to .env: GROQ_API_KEY=your_key_from_console.groq.com

Problem: "npm command not found"
Solution: Install Node.js from https://nodejs.org/

Problem: Frontend shows "Unable to connect to server"
Solution: Backend not running or on wrong port. Check:
  - Backend running: python app.py
  - Port 3000 is free: lsof -i :3000
  - CORS enabled in app.py (already is)
